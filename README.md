# Create Enviroment 

### How to add @angular-devkit/build-angular

Use the package manager npm to see and read more.

```bash
npm install
```

### Run
Go to aspnet-core\src\ProjectManagement.Web.Host
```bash
dotnet run
```
After to angular and open with terminal
```bash
npm start
```
